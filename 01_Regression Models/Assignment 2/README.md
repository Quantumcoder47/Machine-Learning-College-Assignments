# Assignment 2 — Simple House Price Prediction

Summary
- Predict house prices using linear regression. The notebook walks through loading the dataset, exploring statistics, choosing features, training a LinearRegression model, and evaluating predictions.

Dataset
- `housing_price_dataset.csv` — columns:
  Avg. Area Income, Avg. Area House Age, Avg. Area Number of Rooms, Avg. Area Number of Bedrooms, Area Population, Price, Address

How to run
- Open `model.ipynb` and run the cells. Requires pandas, numpy, scikit-learn, matplotlib.

Notes
- The dataset includes address strings; the notebook focuses on numeric predictive features.
