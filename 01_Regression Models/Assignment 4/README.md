# Assignment 4 — Non-Linear Regression: China GDP Prediction

Summary
- Demonstrates polynomial (non-linear) regression to model China's GDP over time. Includes feature preparation (year normalization), testing multiple polynomial degrees, model selection and visualization.

Dataset
- `china_gdp.csv` — columns:
  Year, Value

How to run
- Open `model.ipynb` and run cells. Requires pandas, numpy, scikit-learn, matplotlib.

Notes
- Notebook evaluates multiple polynomial degrees and compares R² and error metrics.
