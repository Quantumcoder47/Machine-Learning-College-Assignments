# Assignment 8 — Naive Bayes: Diabetes Prediction

Summary
- Demonstrates Gaussian Naive Bayes on the Pima Indians Diabetes dataset. Covers dataset loading, feature/target separation, training, evaluation, and simple visualizations.

Dataset
- `pima-indians-diabetes.data.csv` — typical columns (not included in file header): Pregnancies, Glucose, BloodPressure, SkinThickness, Insulin, BMI, DiabetesPedigree, Age, Outcome

How to run
- Open `model.ipynb`. Requires pandas, numpy, scikit-learn, seaborn, matplotlib.

Notes
- The notebook provides named columns within the code when loading this CSV (names passed to read_csv).
