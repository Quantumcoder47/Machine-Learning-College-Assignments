# Assignment 6 — KNN: Telecommunications Customer Classification

Summary
- Applies K-Nearest Neighbors to classify telecom customers into categories. It includes optimal-K selection, scaling, evaluation and visualizations (confusion matrix, per-category accuracy).

Dataset
- `teleCust.csv` — columns:
  region, tenure, age, marital, address, income, ed, employ, retire, gender, reside, custcat

How to run
- Open `model.ipynb`. Requires pandas, numpy, scikit-learn, seaborn, matplotlib.

Notes
- The notebook demonstrates stratified splits and K selection via accuracy curves.
