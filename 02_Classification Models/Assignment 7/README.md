# Assignment 7 — Drug Classification

Summary
- Small classification notebook that predicts drug type given patient features (age, sex, BP, cholesterol, Na_to_K). Demonstrates simple preprocessing and multi-class classification.

Dataset
- `drug.csv` — columns:
  Age, Sex, BP, Cholesterol, Na_to_K, Drug

How to run
- Open `model.ipynb`. Requires pandas, numpy, scikit-learn, matplotlib.

Notes
- Good exercise for encoding categorical features and evaluating multi-class classifiers.
