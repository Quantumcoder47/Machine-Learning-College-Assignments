# Assignment 13 — Customer Segmentation (Clustering)

Summary
- Clustering and segmentation exercises using customer financial data. The notebook experiments with K-Means / hierarchical clustering, explores features, and interprets segments.

Dataset
- `Cust_Segmentation.csv` — columns:
  Customer Id, Age, Edu, Years Employed, Income, Card Debt, Other Debt, Defaulted, Address, DebtIncomeRatio

How to run
- Open `model.ipynb`. Requires pandas, numpy, scikit-learn, seaborn, matplotlib.

Notes
- The notebook includes feature scaling and visualizations to interpret clusters (cluster sizes, characteristics).
